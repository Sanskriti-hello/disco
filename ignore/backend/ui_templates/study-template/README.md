# Study Template

A teal/turquoise themed study application template with summary box, search bar, label buttons, and shapes section using React and TypeScript.

## Features

- Summary box with large text area
- Search bar with icon buttons
- Interactive label buttons with active state
- Shapes display section
- Study-themed header with icons pattern
- Responsive design

## Installation & Running

```bash
npm install
npm start
```

The app will start on `http://localhost:3007` (or the next available port).
