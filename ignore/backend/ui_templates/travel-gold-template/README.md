# Travel Gold Template

A responsive travel-themed template with photo gallery, calendar, menu section, and accordion FAQ panel using React and TypeScript.

## Features

- Photo gallery with 6 items
- Calendar widget with month/year navigation
- Menu section with keyboard shortcuts
- Expandable accordion FAQ section
- Gold/orange color scheme
- Responsive design

## Installation & Running

```bash
npm install
npm start
```

The app will start on `http://localhost:3003` (or the next available port).
